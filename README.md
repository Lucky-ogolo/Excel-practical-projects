# Excel Practical Tools & Automation Projects

This repository contains a set of Excel-based practical projects designed to showcase automation, data organization, and process tracking skills. These projects were created as part of hands-on practice to build a portfolio of realistic business tools suitable for entry-level data analyst roles or internships.

## Projects Included

1. **Mr Biggs Customer Order Form**  
   - A form-based Excel tool to record customer orders efficiently.  
     

2. **Time Tracker**  
   - An Excel tracker for recording and managing work hours
   - - Demonstrates formula usage, date calculations, and structured data entry.

3. **Expense Tracker**  
   - A simple tool to monitor and categorize personal or business expenses.  
   - Includes automated calculations for totals, categories, and summaries.

4. **Simple Budget Tracker**  
   - Tracks income, expenses, and budget allocation.  
   - Highlights formula-based summaries, variance calculation, and clear visual layout.

## Key Skills Demonstrated

- Excel formulas and functions (SUM, IF, DATE calculations, etc.)  
- Data validation and structured input  
- Basic automation (macros where applicable)  
- Dashboarding and reporting for practical business tools  
- Organization of multiple datasets in a single workbook

## Usage

Each project is self-contained in its own file within this repository. Open the Excel files to explore the forms, trackers, and dashboards.  

- **Mr Biggs Customer Order Form:** Fill out the form and submit entries to see how the structured database updates automatically.  
- **Time Tracker, Expense Tracker, Budget Tracker:** Enter sample data to observe calculations and summaries.

## Purpose

These projects are designed to simulate real-world business scenarios, demonstrating the ability to create practical, functional tools that improve data entry, tracking, and reporting efficiency.

## Note

These projects are meant for **learning, portfolio building, and demonstration purposes**. They are ideal for showcasing Excel skills to recruiters or potential employers.
